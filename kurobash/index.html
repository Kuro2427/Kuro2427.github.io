KUROBASH

Kurobash (lit: Kuro's kitbash) is "my" shitty minecraft resource pack! It's basically just all the stuff I like from other packs (mostly UI stuff and other quality of life improvements), without the things I don't like in them compiled into one neat little 42 mb zip file.



See how little it changes?

Features

- Skedgypack's icons



- Default Dark Mode UI



- NewDefault+'s font and other minor components (You're looking at them right now)

- Max's Better Mobs, like, the whole resource pack


AND MORE!
